Entrar no painel

Cole sua chave do GitHub (fica salva só neste navegador). Com ela o painel publica as mudanças sozinho.

Como criar a chave (uma vez só)
  1. Abra github.com/settings/personal-access-tokens/new (logado como zett-romao).
  2. Em Repository access → Only select repositories → escolha drg-marketing.
  3. Em Permissions → Repository permissions → Contents = Read and write.
  4. Clique em Generate token, copie e cole aqui.
